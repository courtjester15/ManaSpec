🧾 ManaSpec — Dev Notes (Alpha State)
🧠 Project Summary

ManaSpec is an MTG speculation trading terminal built with:

Vanilla HTML / CSS / JS
Tabulator (portfolio UI)
Scryfall API (cards + pricing only)
localStorage (state persistence)
Core Purpose

Simulate and track MTG buy/sell positions:

speculation tracking
portfolio P/L
cash management
price-based decision support

Not a collection tracker — a trading tool.

⚙️ CORE SYSTEM (ALPHA COMPLETE)
🧱 Data Model (Spec Object)
{
  id,
  name,
  set_code,
  set_name,
  collector_number,
  qty,
  buyPrice,        // avg cost basis
  currentPrice,    // live snapshot
  pl,
  buyDate,
  foil (planned)
}
💰 Trading Engine
BUY
decreases cash
increases qty
recalculates avg buyPrice
sets buyDate on first purchase
SELL
increases cash at currentPrice
decreases qty
removes entry if qty = 0
DELETE
removes position entirely
📊 Pricing System
Scryfall API provides:
USD current price
periodic refresh updates all holdings
portfolio values recalculated on refresh
🔍 Search Flow
fuzzy search:
/cards/named?fuzzy=
select card → fetch oracle data
fetch printings:
oracleid search
user selects printing → becomes spec entry
📈 Portfolio UI

Tabulator grid includes:

set / collector #
card name
set name
quantity
buy price (avg cost)
current price
P/L ($)
P/L (%)
Summary Bar
cash
invested
portfolio value
total P/L
💾 Persistence

localStorage keys:

specs
cash

Save system:

updates state → writes to storage → refreshes UI
